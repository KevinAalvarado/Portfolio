  • Java
  • NetBeans
  • Oracle
  • Swing

ISSS Salud - App Móvil

Aplicación móvil para Android que facilita a los pacientes la gestión de citas médicas, visualización de resultados y seguimiento de tratamientos. Desarrollada con Kotlin.

  • Kotlin
  • Android Studio
  • Firebase
  • Material Design

Plataforma web para la administración de la flota de transporte del ISSS. Permite programar rutas, gestionar mantenimientos y optimizar el uso de recursos de transporte.

  • CSS

  • Bootstrap
  • Node.js

Aplicación de escritorio desarrollada en C# para la gestión de datos policiales. Incluye registro de incidentes, seguimiento de casos y generación de reportes estadísticos para análisis de criminalidad.
